# 3D-printable-student-office-desk-organization-tools

This is a growing library of useful, 3D-printable desk organization tools designed for students, creators, and anyone who wants a cleaner workspace. Each model is designed to be practical, easy to print, and customizable. Contributions and ideas are always welcome!

## Included Designs
- Monitor-Riser mounted MagSafe phone charger
- LED remote holder
- Drink coaster holder

## How to Use
1. Open the STEP file in your preferred CAD software.  
2. Customize if needed.  
3. 3D print using the STL file if desired.

## Contributing
Feel free to add new designs! Please organize them in their own folder with STEP/STL files and images.
